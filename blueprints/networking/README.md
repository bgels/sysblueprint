# Networking Lab Blueprints

This directory contains blueprints for network infrastructure and protocol laboratories.

## Available Blueprints

### Basic LAN Lab
**File:** `basic-lan-lab.md`

A foundational local area network laboratory for learning network fundamentals, testing configurations, and developing network applications.

**Key Topics:**
- Network protocol analysis
- DHCP and DNS configuration
- Routing and NAT
- Network troubleshooting

**Difficulty:** Beginner
**Setup Time:** 1-2 hours
**Resources Required:** 1 router, 1 switch, 1 server, 3+ workstations

## Prerequisites

Before starting any networking lab blueprint:

1. **Basic Networking Knowledge**
   - Understanding of IP addressing and subnetting
   - Familiarity with OSI model layers
   - Basic command line skills

2. **Hardware or Virtualization**
   - Physical network equipment OR
   - Virtualization platform (VMware, VirtualBox, etc.)
   - GNS3 or EVE-NG for network simulation

3. **Software Tools**
   - SSH client
   - Network analysis tools (Wireshark, tcpdump)
   - Terminal emulator
